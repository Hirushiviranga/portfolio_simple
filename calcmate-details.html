CalcMate

Calculator Web Application: CalcMate is a functional utility tool designed to handle fundamental arithmetic operations with high precision.

Key Features:

  • Core Operations: Seamless addition, subtraction, multiplication, and division.
  • History Tracking: A dedicated page to review and audit past calculations.
  • QA Practice: Developed as a benchmark project for applying software testing and quality assurance principles.

The application features a clean, professional interface that prioritizes speed and accuracy, built using a modern decoupled architecture.
