Day Pilot (WPF)

Productivity Desktop Application: This is a modern, intuitive productivity tool built using WPF (Windows Presentation Foundation) and SQLite for local data persistence.

Core Features:

  • Task Management: Full CRUD operations to add, update, and organize daily tasks.
  • Calendar Integration: A seamless interface to schedule and track long-term events.
  • Modern UI: A clean navy blue and orange palette designed for high user engagement and clarity.

This project showcases my ability to integrate local databases with desktop GUIs to create functional, user-centric software.
