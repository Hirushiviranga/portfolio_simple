DayPilot

Productivity Web Application: DayPilot is a full-stack platform designed to streamline personal task management. It integrates a robust backend to handle real-time data for reminders, to-do lists, and calendar events.

The frontend features a glassmorphism aesthetic with a navy blue theme, ensuring high accessibility and a modern feel. By leveraging XAMPP for MySQL database management, the app provides reliable data persistence for all user activities.
