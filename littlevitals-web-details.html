LittleVitals Web

Baby Management System - MERN Stack: LittleVitals is a comprehensive full-stack web application designed for both parents and healthcare providers to manage infant records seamlessly.

The system features a multi-role architecture:

  • Admin: Full CRUD control over medical and clinical records.
  • User (Parent): Secure dashboard to track nutrition, BMI trends, and doctor reviews.

Built with a focus on visual analytics and secure authentication, it simplifies health tracking through modern UI components and real-time data processing.
