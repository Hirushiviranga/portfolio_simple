Automated Cloud Deployment

CI/CD Pipeline & Infrastructure: This project showcases a fully automated Continuous Integration and Continuous Deployment (CI/CD) architecture designed to host a full-stack web application with zero manual intervention.

The infrastructure focuses on scalability and automation:

  • Cloud Hosting: Containerized frontend (React/Nginx) and backend (Node.js) applications deployed on an optimized AWS EC2 instance, utilizing custom swap memory and storage management.
  • Pipeline Automation: A robust Jenkins Declarative Pipeline that listens for GitHub webhooks to automatically build, test, and push Docker images to Docker Hub before executing a live deployment.

Built with modern DevOps principles, this setup demonstrates end-to-end cloud orchestration, secure network configurations via AWS Security Groups, and efficient container management.
