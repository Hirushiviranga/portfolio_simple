TripiGO (MERN Stack)

Integrated Travel Management System: A comprehensive web platform built to streamline travel experiences, featuring secure user authentication and real-time booking persistence using the MERN stack.

Core Features:

  • Secure Authentication: JWT-based login and signup system protecting user-specific data.
  • Unified Bookings: Integrated modules for vehicle rentals and destination stays with automated database updates.
  • Dynamic Dashboard: A personalized "My Bookings" chart with real-time status tracking (Confirmed/Rejected) pulled from MongoDB.

This project demonstrates my proficiency in building scalable RESTful APIs and connecting them to intuitive React frontends, ensuring high data integrity and a seamless user journey.
